### 🚀 Əsas Layihəm: StockPrice Microservice System
Bu layihə bir neçə mikroservisdən (Gateway, Config, Service Discovery, Notification) ibarət kompleks sistemdir. Bütün komponentləri bir yerdə görmək üçün təşkilat profilinə keçid edə bilərsiniz:
👉 **[StockPrice-Microservice Təşkilatı](https://github.com/StockPrice-Microservice)**

### 🛠 Texniki Bacarıqlar (Tech Stack)
- **Dillər:** Java
- **Frameworks:** Spring Boot, Spring Cloud (Eureka, Config Server, OpenFeign, API Gateway)
- **Məlumat Bazaları:** PostgreSQL, MySQL, Redis (Caching)
- **Mesajlaşma:** RabbitMQ
- **Alətlər:** Docker, MinIO, Web Scraping (JSoup/Selenium)
